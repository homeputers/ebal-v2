title: Crear una lista de canciones
description: Arma una lista de adoración con canciones y arreglos.

1. Navega al servicio próximo en el panel.
2. Agrega canciones o arreglos en el orden deseado.
3. Incluye notas sobre transiciones, tonalidades o instrumentación para mantener alineado al equipo.

A medida que se publiquen nuevas funciones, esta guía se ampliará con capturas de pantalla y consejos avanzados.
