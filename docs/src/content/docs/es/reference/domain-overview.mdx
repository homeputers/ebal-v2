---
title: Panorama del dominio
description: Describe cómo Ebal modela miembros, canciones, arreglos y servicios.
---

Ebal v2 modela entidades clave de planificación de adoración como miembros, grupos, canciones, arreglos y planes de servicio. Comprender estas relaciones ayuda a los administradores a mantener los datos limpios y habilitar reportes.

A medida que la plataforma evolucione, se publicarán aquí esquemas más detallados y referencias de API.
