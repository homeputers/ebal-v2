# Documentación de Ebal v2

Bienvenido a la base de conocimiento viva de Every Breath And Life. Este sitio reúne los manuales, guías prácticas y referencias técnicas que necesitas para planificar servicios con confianza.

> **Nota:** Estos documentos están en progreso. Usa la barra lateral o la búsqueda para ir directo a un tema.

## ¿Qué es Ebal v2?

Ebal v2 es la siguiente iteración de nuestra herramienta de planificación para equipos de adoración. Ayuda a los líderes a coordinar la selección de canciones, arreglos, notas de ensayo y orden de servicio para que los equipos puedan enfocarse en el ministerio.

## Primeros pasos

- Revisa la sección de **Manuales** para profundizar en características y flujos de trabajo.
- Usa las **Guías prácticas** para respuestas rápidas a tareas comunes.
- Consulta las páginas de **Referencia** para detalles de configuración y conceptos del dominio.

Si ves algo faltante, abre un issue o PR para seguir mejorando esta documentación en conjunto.
