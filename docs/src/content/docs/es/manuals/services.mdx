# Servicios

## Descripción general
El espacio de Servicios te permite trazar toda una reunión de adoración. Construye un plan agregando canciones, lecturas y notas, ordena cada elemento hasta lograr un flujo significativo y genera vistas tanto para compartir como para imprimir con tu equipo.

## Requisitos previos
- Rol de Administrador o Planificador para crear y modificar servicios.
- Canciones y arreglos ya cargados en tu catálogo para los elementos musicales.

## Paso a paso
1. **Crea el servicio.** Desde la lista de Servicios, haz clic en **Nuevo servicio**, define la fecha/hora de inicio y la ubicación opcional, y guarda.
2. **Filtra la agenda.** Usa los filtros de búsqueda y fecha para encontrar rápidamente servicios existentes.
3. **Abre el constructor de planes.** Selecciona un servicio para ir a su página de detalle y utiliza la columna izquierda para agregar elementos.
4. **Agrega elementos.** Elige el tipo de elemento:
   - **Canción:** Elige una canción, selecciona un arreglo, revisa la vista previa del resumen de tonalidad y agrega notas opcionales.
   - **Lectura o nota:** Ingresa notas de texto libre (referencias bíblicas, transiciones, anuncios).
5. **Organiza el flujo.** Arrastra el asa de cada elemento para reordenarlo. Los cambios se guardan automáticamente en segundo plano.
6. **Edita notas en el mismo lugar.** Actualiza el campo de notas de cualquier elemento; sal del área de texto para guardar tus cambios.
7. **Administra el servicio.** Usa los botones de acción para editar los metadatos, quitar elementos o eliminar el servicio cuando ya no sea necesario.
8. **Comparte el plan.** Haz clic en **Vista de plan** para abrir una página de solo lectura (la URL incluye un token `share`) que los líderes pueden distribuir.
9. **Imprime la hoja de ruta.** Selecciona **Imprimir** para usar un diseño optimizado y activar el cuadro de diálogo de impresión del navegador.

> _Marcador de posición de captura: constructor del plan de servicio._
> _Marcador de posición de captura: plan de servicio imprimible._

## Ejemplos
- Diseña una reunión dominical con canciones, una lectura bíblica, una nota de comunión e instrucciones de bendición en elementos separados del plan.
- Comparte el enlace del plan con los voluntarios para que puedan ver el orden sin iniciar sesión (el token de uso compartido controla el acceso).

## Consejos y buenas prácticas
- Agrega notas que describan las transiciones (por ejemplo, "Vamp acústico hacia la oración") para que todos sepan qué ocurre entre canciones.
- La vista previa de la canción muestra tonalidad, tempo y métrica del arreglo; úsala para confirmar que elegiste la versión correcta.
- Mantén el arrastrar y soltar ordenado colapsando las secciones que ya terminaste antes de pasar al siguiente bloque.
- Usa la vista de impresión del plan durante el ensayo para capturar anotaciones de último minuto directamente en papel.
