# Conjuntos de canciones

## Descripción general
Los conjuntos de canciones te permiten agrupar arreglos en secuencias reutilizables para noches de adoración, servicios juveniles o reuniones recurrentes. Cada elemento registra ajustes de transposición y cejilla, así que puedes añadir el conjunto a un servicio con plena confianza.

## Requisitos previos
- Rol de Administrador o Planificador para gestionar conjuntos de canciones.
- Canciones y arreglos preparados en el catálogo.

## Paso a paso
1. **Crea un conjunto.** Desde la lista de Conjuntos de canciones, haz clic en **Nuevo conjunto**, asígnale un nombre (por ejemplo, «Apertura noche de jóvenes») y guarda.
2. **Abre el constructor de conjuntos.** Selecciona el conjunto para ver su página de detalle con las herramientas de agregar y ordenar.
3. **Elige una canción.** Usa el buscador de canciones para encontrar el título que necesitas.
4. **Escoge un arreglo.** Selecciona el arreglo específico; una línea de vista previa muestra título, tonalidad, tempo y métrica.
5. **Configura transposición y cejilla.** Introduce los semitonos de cambio (−6 a +6) y la posición de cejilla (0–7). El resumen de tonalidad se actualiza para reflejar la tonalidad sonora y la forma.
6. **Agrega el elemento.** Haz clic en **Agregar al conjunto** para añadirlo a la lista en orden.
7. **Reordena los elementos.** Arrastra el asa junto a cualquier elemento para reorganizar el conjunto; los cambios se guardan de inmediato.
8. **Ajusta elementos existentes.** Modifica los controles deslizantes de transposición/cejilla o elimina canciones en línea. Activa el selector de bemoles/sostenidos en la parte superior para ver las tonalidades como prefiera tu equipo.
9. **Edita los detalles del conjunto.** Cambia el nombre desde el menú de acciones si el enfoque del conjunto evoluciona con el tiempo.

> _Marcador de posición de captura: constructor de conjuntos de canciones._

## Ejemplos
- **Vista previa de etiqueta:** Un elemento puede mostrar «Living Hope — Tonalidad E • 72 BPM • 4/4» con un resumen «Tonalidad E → F (+1), Cejilla 2 (formas en D)».
- **Reutilización estacional:** Crea una vez un conjunto «Mañana de Adviento» y añádelo a varios servicios de diciembre sin volver a configurar las tonalidades.

## Consejos y buenas prácticas
- Usa la opción de bemoles al planificar para secciones de metales; así confirman de inmediato las equivalencias enarmónicas.
- Mantén los ajustes de transposición dentro de rangos cómodos (el editor limita a ±6 semitonos para ayudarte).
- Usa los conjuntos como punto de partida para los servicios: copia el orden de canciones y luego agrega lecturas o notas por servicio.
- Elimina pronto los elementos que no utilices para que la caché de arreglos se mantenga enfocada en las partituras que realmente usas.
