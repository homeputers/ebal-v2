# Miembros y grupos

## Descripción general
Las áreas de Miembros y Grupos te ayudan a gestionar a todas las personas involucradas en el ministerio de adoración y asignarlas a los equipos con los que sirven. Usa estas páginas para mantener actualizados los datos de contacto, registrar instrumentos y crear grupos enfocados para la programación.

## Requisitos previos
- Rol de Administrador o Planificador para crear, editar o eliminar miembros y grupos.
- Registros de miembros para cada persona que planeas asignar a un grupo.

## Paso a paso
1. **Revisa la lista de miembros.** Usa la barra de búsqueda para filtrar por nombre o instrumento y navega por los resultados si administras un elenco grande.
2. **Agrega un nuevo miembro.** Selecciona **Nuevo miembro**, completa el nombre para mostrar, instrumentos (separados por comas), información de contacto opcional y fecha de cumpleaños, y guarda para crear el perfil.
3. **Actualiza o elimina miembros.** Abre las acciones contextualizadas junto a cada miembro para editar sus detalles o eliminar el registro de forma permanente (tras confirmar el aviso).
4. **Crea un grupo.** En la página de Grupos, elige **Nuevo grupo**, nombra el equipo (por ejemplo, «Vocalistas») y guarda.
5. **Asigna personas a un grupo.** Abre un grupo, busca a un miembro con el selector predictivo, haz clic en **Agregar** y repite con cada participante. Quita miembros con el botón **Quitar** cuando dejen el equipo.
6. **Mantén las listas al día.** Edita los nombres de los grupos en línea o ajusta la información de los miembros en cualquier momento. Los cambios se sincronizan automáticamente con el resto de la aplicación.

> _Marcador de posición de captura: lista de miembros._
> _Marcador de posición de captura: detalle de grupo._

## Ejemplos
- **Higiene del elenco:** Agrega instrumentos como `Voz, Guitarra acústica` para que los planificadores filtren rápidamente.
- **Asignaciones por grupo:** Crea grupos separados como «Banda», «Voces» y «Técnica» para facilitar la planificación de servicios.

## Consejos y buenas prácticas
- Usa nombres de instrumentos consistentes (por ejemplo, «Guitarra eléctrica» en lugar de variantes) para que la búsqueda sea predecible.
- Captura correo electrónico y teléfono para enviar recordatorios desde un solo lugar.
- Ingresa mes y día de cumpleaños para planear celebraciones o programar alrededor de fechas clave.
- Audita periódicamente los grupos para quitar miembros inactivos y mantener precisas las comunicaciones de ensayo.
